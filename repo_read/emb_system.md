| 模块 | 功能说明 | 项目对应/举例 | 关键接口/注意事项（选型 & 实现要点） |
|---|---|---|---|
| MCU / VCU（核心控制器） | 负责整车底层控制：驱动器组网管理、速度/扭矩指令下发、状态机（上电-预充-使能-运行-故障-安全停机）、故障处理与日志 | 保存：轮子方向标定、驱动器ID映射、限流/加速度参数等 | 关键接口：CAN/RS485、GPIO急停、PWM/ADC；注意：硬实时优先，通信容错、看门狗、掉电保护 |
| Flash / SRAM | Flash存固件、参数与标定值；SRAM运行时数据（协议栈缓冲、状态机、控制环数据） | 保存：轮子方向标定、驱动器ID映射、限流/加速度参数等 | 注意：参数区磨损均衡；掉电写保护；日志与参数分区 |
| 外设（GPIO/UART/SPI/I2C/ADC/PWM） | GPIO用于急停、驱动器使能、故障输入、限位；UART用于调试/上位机或部分设备；SPI/I2C接IMU等；ADC用于电压电流采样；PWM用于舵机/电机控制 | IMU/里程计/传感器、驱动器使能、急停、限位开关 | 注意：ADC采样抗噪声；PWM频率与分辨率；I2C上拉与线长；SPI走线与屏蔽 |
| 轮驱电机（BLDC Traction Motor） | 将电能转换为轮端机械功；提供霍尔/编码器反馈以实现低速稳定与闭环控制 | 48V三相BLDC（暂定选型方向） | 注意：霍尔/编码器供电与抗干扰；温升与沙尘防护 |
| 轮驱减速机构（Planetary Gearbox） | 降速增扭，匹配轮速与驱动力；与轮端机构集成以适应沙地重载与冲击 | 行星减速箱PLF/PZF系列（对比） | 注意：冲击载荷与间隙；密封与润滑；安装同轴度 |
| 轮驱驱动器/伺服驱动（FOC Driver / Servo Drive） | 接收VCU指令（速度/扭矩/位置），输出三相UVW驱动电机；实现电流环/速度环（可选位置环）；提供过流/过温/欠压等保护 | CAN/CANopen/RS485驱动器 | 注意：指令接口与控制周期；故障码处理；急停硬切断 |
| 轮腿抬升执行器（Joint Servo Actuator） | 实现1DOF抬腿（主动悬挂/跨障）；与轮驱配合实现“抬起-落下-软着陆-限扭矩” | MF9025×8 | 注意：供电与瞬态电流；关节限位；结构强度 |
| 传感器（状态+环境+运动） | 用于控制与安全：姿态IMU、轮速/里程计、温度、电流、电压；可选环境传感（温湿度/光照）用于作业评估 | IMU/里程计/红外传感 | 注意：安装位置；标定；滤波；抗沙尘与遮挡 |
| 通信模块（车内总线 + 远程链路） | 车内：VCU与8个驱动器/伺服的实时通信；车外：与上位机/遥控/上层计算单元通信 | CAN / RS485(Modbus) / WiFi | 注意：总线终端电阻；拓扑；协议容错；日志 |
| 电源管理（低压电源树） | 将48V主电转换为24V/12V/5V，供给控制器、伺服、传感器；监控电压并在异常时安全降级/停机 | 48V→24V/12V/5V | 注意：DC-DC裕量；热设计；EMI；防反接 |
| 高压/大电流配电与安全 | 48V主回路的安全与上电流程：主保险、接触器、预充、支路保险、急停硬切断，保证故障时可控停机 | 急停、接触器、预充电阻 | 注意：预充时间常数；故障短路保护；急停回路硬件优先 |
| 线束/连接器/防护 | 将电源、UVW、反馈、总线可靠地连接到每个轮模块；提供防尘、防水、抗振与应力释放 | 线束设计与连接器选型 | 注意：IP等级；拉力释放；屏蔽接地；耐磨 |
| 软件架构 | 通信协议栈、状态机、速度控制、限流/限扭矩、轮间同步、故障处理、日志与参数管理 | bring-up台架、8轮一致性、故障注入 | 注意：分层架构；日志与参数；单元测试 |
| 调试与测试 | 单轮台架验证、8轮联调、故障注入测试（欠压/过流/断线）、温升测试、沙尘环境可靠性检查 | 单轮→多轮复制验证 | 注意：测试记录；版本管理；可复现性 |
